Locks a random player slot when players take damage, also has a shared mode where all players share locked slots.


## Installation
1. Download the data pack.
2. Place the file in the `datapacks` folder of your Minecraft world.
3. Run `/reload` in-game.

## Available Commands:
- `/trigger share_locks` Enables/Disables locked slots sharing.
- `/trigger friendly_fire` Enables/Disables friendly fire (PVP) between players.
- `/trigger reset` Reset all locked slots for all players.

## Requirements
- Minecraft 1.21.x or higher.
